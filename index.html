FAQ(Często Zadawane Pytania)

  • Dlaczego wyświetla się kilka takich samych stacji i/lub niektóre z nich nie działają?
  • Wynika to z błędów API. API jest otwarte i darmowe, więc każdy może dodać do niego stację radiową. Po prostu takie stacje zostały dodane dawno i/lub zostały tymczasowo wyłączono. Aktualnie pracujemy nad usunięciem z listy winików wyszukiwania stacji które nie działają.
